# OSX 系统开机自动执行脚本抓取数据

## 使用：

1、plist文件可以放置的位置说明：
      /Library/LaunchDaemons 系统启动，就会执行
     /Library/LaunchAgents 用户登录后才执行

2、脚本\sh文件位置：
     /usr/local/

详情参见可以参见文章：[传送门](www.3code.info/2016/11/08/pythonGetFund/)
